# 🌳 Vidguk Manifesto

A family tree is not a social network.

It should never depend on the existence of a particular company.

Family history should not disappear because a server is shut down or a service changes its rules.

Data belongs to the individual and their family.

Vidguk uses open formats and open source software so that family archives remain accessible regardless of future technologies.

If someone opens a family archive one hundred years from now, they should be able to read it without subscriptions, cloud services, or anyone's permission.

We are not building a data collection platform.

We are building a tool for preserving memory.

---

**Vidguk** — a memory tool that leaves data in the hands of people.
