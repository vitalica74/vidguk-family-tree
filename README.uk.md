# 🌳 Vidguk — Родове дерево

> **Історія вашого роду належить вашій родині, а не хмарному сервісу.**

Vidguk — це PWA-застосунок для створення, збереження та передачі родового дерева.

Основна ідея проєкту — користувач повинен повністю контролювати свої дані.

## Можливості

- 🌳 Родове дерево
- 🌿 Імпорт GEDCOM
- 📦 ZIP резервні копії
- 🔍 Пошук дублікатів
- 💾 Автоматичний локальний резерв
- 📱 PWA
- 🌐 Робота без Інтернету
